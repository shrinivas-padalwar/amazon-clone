# Amazon Clone

A front-end clone of the Amazon homepage built with HTML and CSS.

## Features
- Amazon-style navigation bar
- Hero/banner section
- Product category grid
- Footer

## Tech Stack
- HTML5
- CSS3 (Flexbox)
